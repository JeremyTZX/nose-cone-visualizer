# Interactive Nose Cone Visualizer

## Purpose
This version prompts the user for nose cone parameters at runtime, enabling an interactive exploration of the nose cone geometries.

## How to Run
1. Open `Nose_Cone_Visualizer-Interactive.m` in MATLAB.
2. Run the script
3. Enter parameters as prompted
4. The nose cone plot will be generated automatically based on your input

## Supported Geometries
- Conical
- Parabolic
- Ogive
